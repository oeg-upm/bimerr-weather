# Weather Ontology
This repository contains the code and documentation generated for the Weather ontology which is available at:
http://bimerr.iot.linkeddata.es/def/weather


## Requirements Specification

The following table shows the ORSD (Ontological Requirements Specification Document) that specifies the functional and non-functional requirements for this ontology.
